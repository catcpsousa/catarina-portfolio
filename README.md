# Catarina Sousa Portfolio

This is my personal portfolio, built with HTML, CSS, and JavaScript.

## ✨ Features
- Responsive layout for desktop, tablet, and mobile
- Sliding mobile menu with soft overlay
- Sections: Hero, Projects, Services, About, Contact
- Project filtering by category
- "Back to top" button

## 🚀 How to use
1. Clone or download this repository
2. Open the `index.html` file in your browser

## 📁 Structure
- `index.html` — Main page
- `styles.css` — Global styles
- `js/main.js` — Interaction logic (menu, filters, scroll)
- `projects/` — Project pages and assets

## 🛠️ Notes
- No backend or frameworks required
- Optimized for showcasing real projects
